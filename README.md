## 🚀 Task 1: Lingua AI – AI Language Translation Tool

### Description
Lingua AI is an AI-powered language translation web application developed using React and Claude AI API. The application provides accurate translations across multiple languages with additional intelligent features.

### Features
- Translation across 20+ languages
- Automatic language detection
- Confidence score display
- Text-to-speech functionality
- Translation history
- Copy translated text
- Language swap option

### Technologies Used
- React.js
- JavaScript
- Claude AI API
- Web Speech API

## 🤖 Task 2: NexBot – AI FAQ Chatbot

### Description
NexBot is an AI-powered FAQ chatbot built using React and a custom NLP engine. It understands user queries and retrieves relevant answers using TF-IDF vectorization and cosine similarity.

### Features
- Natural language query processing
- TF-IDF vectorization
- Cosine similarity matching
- Intent detection
- Confidence scoring
- Real-time FAQ search
- Suggested questions

### Technologies Used
- React.js
- JavaScript
- TF-IDF
- Cosine Similarity
- NLP Techniques
